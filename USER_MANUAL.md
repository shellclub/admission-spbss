# คู่มือการใช้งานระบบรับสมัครนักศึกษา
## มหาวิทยาลัยการกีฬาแห่งชาติ

---

### หมายเหตุ:
- ระบบมีการป้องกัน Brute-force: หากกรอกรหัสผิดเกิน 10 ครั้ง จะถูกล็อค 15 นาที
- รหัสผ่านถูกเข้ารหัสด้วย bcrypt เพื่อความปลอดภัย

---

## 2. หน้า Dashboard

หลังเข้าสู่ระบบสำเร็จ จะพบหน้า Dashboard แสดงการ์ดสรุปข้อมูล:
- จำนวนผู้สมัครทั้งหมด
- สถานะการสมัคร (รอสัมภาษณ์/ผ่าน/ไม่ผ่าน)

---

## 3. การจัดการข้อมูลผู้สมัคร

### ดูรายชื่อผู้สมัคร:
1. คลิกเมนู **"เพิ่ม/ดูข้อมูล"** ที่แถบด้านซ้าย
2. ระบบจะแสดงตารางรายชื่อผู้สมัครทั้งหมด

### ค้นหาผู้สมัคร:
- พิมพ์ชื่อหรือเลขที่สมัครในช่องค้นหา

### แก้ไขข้อมูล:
1. คลิกไอคอนดินสอ (✏️) ที่แถวข้อมูล
2. แก้ไขข้อมูลในฟอร์ม
3. กดบันทึก

### ลบข้อมูล:
1. ติ๊กเลือกผู้สมัครที่ต้องการลบ
2. กดปุ่ม **"ลบ"**
3. ยืนยันการลบ

---

## 4. การตั้งค่าบัญชีผู้ใช้

### เปลี่ยนชื่อ/รูปโปรไฟล์:
1. คลิกชื่อผู้ใช้มุมขวาบน
2. เลือก **"แก้ไขข้อมูล"**
3. แก้ไขชื่อหรือคลิกรูปเพื่ออัปโหลดรูปใหม่
4. กดบันทึก

### เปลี่ยนรหัสผ่าน:
1. คลิกชื่อผู้ใช้มุมขวาบน
2. เลือก **"เปลี่ยนรหัส"**
3. กรอกรหัสผ่านปัจจุบันและรหัสใหม่
4. กดบันทึก

---

## 5. การใช้งาน Docker

### เปิดใช้งานระบบ:
```bash
docker-compose -f docker-compose.prod.yml up -d
```

### ปิดระบบ:
```bash
docker-compose -f docker-compose.prod.yml down
```

### ดู Logs:
```bash
docker logs nextjs_prod_app --tail 100
```

---

## 6. การแก้ปัญหาเบื้องต้น

| ปัญหา | วิธีแก้ไข |
|-------|----------|
| เข้าระบบไม่ได้ | ตรวจสอบ username/password, รอ 15 นาทีหากถูกล็อค |
| หน้าเว็บไม่แสดง | ตรวจสอบ Docker container ว่ารันอยู่หรือไม่ |
| ข้อมูลไม่อัปเดต | Refresh หน้าเว็บหรือล้าง Cache |
